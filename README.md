# Penaltı Atışı Oyunu

Modern, animasyonlu ve bağımlılıksız bir tarayıcı oyunu.

## Çalıştırma

Projeyi indirdikten sonra iki farklı şekilde açabilirsiniz:

1. **Doğrudan açma:** `index.html` dosyasına çift tıklayın.
2. **Yerel sunucu ile açma:** Terminalde proje klasörüne girip aşağıdaki komutu çalıştırın:

```bash
npm start
```

Ardından tarayıcıdan `http://localhost:4173` adresine gidin.

## Kontroller

- Sol: `←` veya `A`
- Orta: `↑`, `↓`, `W` veya `S`
- Sağ: `→` veya `D`
- Sıfırla: `R`

## Kontrol Komutları

JavaScript sözdizimini kontrol etmek için:

```bash
npm run check
```
